Giới thiệu:

file chính : index.html chứa tất cả nội dung cần chạy
css: phân loại các đoạn mã css của từng phần giúp dễ quản lý, tìm kiếm, chỉnh sửa
    banner.css: chứa nội dung css của phần Larger Banner
    instagram.css: chứa nội dung css của phần Gallery
    login-cart.css: chứa nội dung css của phần Login và Side Cart
    new-product.css: chứa nội dung css của phần New Product
    side-bar.css: chứa nội dung css của phần Side Bar: Social, Collapsed, Recently Viewed
    style.css: chứa nội dung css chung (body, ...) , Header và Footer
    sub-banner.css: chứa nội dung css của phần Sub Banner
js: 
    Tương tự như folder css
    Chứa các phần code javascript để hỗ trợ tạo hiệu ứng cho web
    Có kèm cmt giải thích
